TorusGeometry

`TorusGeometry` is a class that represents a torus-shaped 3D geometry. A torus is a surface of revolution generated by revolving a circle in three-dimensional space about an axis coplanar with the circle. The TorusGeometry class allows you to create torus geometry with various parameters, providing flexibility and control over its appearance.

TorusGeometry(radius : Float, tube : Float, radialSegments : Integer, tubularSegments : Integer, arc : Float);

radius (float)

The overall radius of the torus, from the center of the torus to the center of the tube.

  • radius: Radius of the torus, from the center of the torus to the center of the tube. Default is 1.
tubeRadius (Float)

The radius of the tube (the thickness of the torus).

  • tube :Radius of the tube. Default is 0.4.
radialSegments (integer)

The number of segmented faces around the circumference of the torus.

  • radialSegments :Default is 12
tubularSegments (Integer)

The number of segmented faces along the tube's circumference.

  • tubularSegments : Default is 48.
arc (Float)

The central angle of the torus. The default is Math.PI * 2, which creates a complete torus.

  • arc : Central angle. Default is Math.PI * 2.

TorusGeometry in Three.js provides a versatile way to create torus-shaped 3D geometries for use in your web-based 3D graphics applications. It offers various parameters to customize the torus's appearance and can be combined with different materials for enhanced visual effects.

Explanation

Importing Libraries and Setting Up the Scene

The first part of the code imports the necessary libraries: Three.js, OrbitControls, and dat.GUI. These libraries are used to create the 3D scene, handle camera controls, and provide a graphical user interface (GUI) for adjusting the torus's parameters.

Creating the Torus and Adding It to the Scene

The code then creates a Three.js scene, a perspective camera, and a WebGL renderer. It also sets up a torus geometry using the TorusGeometry class with specific parameters such as radius, tube radius, radial segments, tubular segments, and arc. The torus is then combined with a material to create a mesh, and the mesh is added to the scene.

GUI Controls for Torus Parameters

A dat.GUI instance is created to provide a user interface for adjusting torus parameters dynamically. A folder named 'Torus Settings' is added to the GUI, and sliders are included for radius, tube radius, radial segments, tubular segments, and arc. The `onChange` event for each parameter triggers a function, updateTorusGeometry, which updates the torus geometry based on the modified parameters.

Animation Loop and Torus Rotation

The animate function is responsible for the animation loop. This function is called repeatedly using requestAnimationFrame, ensuring smooth animation at a consistent frame rate. Within the animation loop, the torus is continuously rotated along its x and y axes, creating an animated effect.

Camera Positioning and Orbit Controls

The camera is positioned at a distance of 10 units along the z-axis, providing a good viewing angle for the torus. OrbitControls are set up to enable user interaction for rotating and zooming the camera around the scene, enhancing the interactive 3D experience.

Handling Window Resize

An event listener is added to handle window resize events. When the window is resized, the camera's aspect ratio is adjusted to match the new window dimensions. This ensures that the rendered scene remains proportional and avoids distortion.

Starting the Animation

Finally, the animate function is called at the end of the code, initiating the animation loop. This continuously updates the torus's rotation and renders the updated scene, resulting in a smooth 3D animation of a rotating torus.
