CylinderGeometry

`CylinderGeometry` is a class that represents a cylinder geometry. It is used to create 3D cylinder shapes, which can be rendered in a Three.js scene.

CylinderGeometry(radiusTop : Float, radiusBottom : Float, height : Float, radialSegments : Integer, heightSegments : Integer, openEnded : Boolean, thetaStart : Float, thetaLength : Float)

radiusTop (Float)

The radius of the cylinder at the top. This parameter defines the size of the circular cross-section at the upper end of the cylinder. If radiusTop is set to 0, the top will be flat.

  • radiusTop: The radius of the cylinder at the top. Default is 1.
radiusBottom (Float)

The radius of the cylinder at the bottom. Similar to radiusTop, this parameter defines the size of the circular cross-section at the lower end of the cylinder. If radiusBottom is set to 0, the bottom will be flat.

  • radiusBottom: The radius of the cylinder at the bottom. Default is 1.
height (Float)

The overall height of the cylinder. It represents the distance between the top and bottom faces of the cylinder.

  • height: The height of the cylinder. Default is 1.
radialSegments (Integer)

The number of segmented faces around the circumference of the cylinder. Increasing this value will result in a smoother appearance but may impact performance. This parameter controls how many triangular faces approximate the circular cross-section.

  • radialSegments:The number of segmented faces around the circumference of the cylinder. Default is 8.
heightSegments (Integer)

The number of segmented faces along the height of the cylinder. Similar to radialSegments, increasing this value leads to a smoother appearance but can affect performance. It controls how many vertical divisions the cylinder is split into.

  • heightSegments:The number of segmented faces along the height of the cylinder. Default is 1.
openEnded (Boolean)

A boolean indicating whether the ends of the cylinder are open (true) or capped (false). If openEnded is true, the cylinder will not have top and bottom faces, resulting in a hollow tube. If false, the cylinder will be closed at both ends.

  • openEnded: A boolean indicating whether the ends of the cylinder are open or capped. Default is false (capped).
thetaStart (Float)

The start angle for the first segment, in radians. It defines the rotation of the starting point of the cylinder geometry. By default, this is set to 0.

  • thetaStart:The start angle for the first segment, in radians. Default is 0.
thetaLength (Float)

The central angle of the circular sector, in radians. This parameter determines how much of the cylinder is drawn. A value of 2 * Math.PI creates a complete cylinder, while a smaller value creates a partial cylinder.

  • thetaLength:The central angle of the circular sector, in radians. Default is 2 * Math.PI, which creates a complete cylinder.

These parameters give you fine control over the dimensions, smoothness, and appearance of the cylinder geometry you create using CylinderGeometry in Three.js. Adjusting these parameters allows you to create a wide variety of cylinder shapes to suit your 3D scene.

Three.js SphereGeometry Example

    
// Import necessary Three.js modules
import * as THREE from 'three';
import './style.css'
import * as dat from 'dat.gui';

// Create a scene
const scene = new THREE.Scene();

// Set up dat.GUI
const gui = new dat.GUI();

// Create an object to store parameters
const cylinderParams = {{'{'}}
    radiusTop: 1,
    radiusBottom: 1,
    height: 2,
    radialSegments: 16,
    heightSegments: 1,
    openEnded: false,
    thetaStart: 0,
    thetaLength: 2 * Math.PI,
    {{'}'}};

// Create a cylinder geometry
const cylinderGeometry = new THREE.CylinderGeometry(
    cylinderParams.radiusTop,
    cylinderParams.radiusBottom,
    cylinderParams.height,
    cylinderParams.radialSegments,
    cylinderParams.heightSegments,
    cylinderParams.openEnded,
    cylinderParams.thetaStart,
    cylinderParams.thetaLength
);

// Create a material
const material = new THREE.MeshBasicMaterial({{'{'}} color: 0x00ff00, wireframe: true {{'}'}});

// Create a mesh
const cylinderMesh = new THREE.Mesh(cylinderGeometry, material);

// Add the mesh to the scene
scene.add(cylinderMesh);

// Function to update the cylinder geometry based on parameters
function updateCylinder() {{'{'}}
    scene.remove(cylinderMesh);

    const updatedGeometry = new THREE.CylinderGeometry(
        cylinderParams.radiusTop,
        cylinderParams.radiusBottom,
        cylinderParams.height,
        cylinderParams.radialSegments,
        cylinderParams.heightSegments,
        cylinderParams.openEnded,
        cylinderParams.thetaStart,
        cylinderParams.thetaLength
    );

    cylinderMesh.geometry.dispose();
    cylinderMesh.geometry = updatedGeometry;

    scene.add(cylinderMesh);
    {{'}'}}

// Add GUI controls
gui.add(cylinderParams, 'radiusTop', 0.1, 3).onChange(updateCylinder);
gui.add(cylinderParams, 'radiusBottom', 0.1, 3).onChange(updateCylinder);
gui.add(cylinderParams, 'height', 0.1, 5).onChange(updateCylinder);
gui.add(cylinderParams, 'radialSegments', 3, 32, 1).onChange(updateCylinder);
gui.add(cylinderParams, 'heightSegments', 1, 10, 1).onChange(updateCylinder);
gui.add(cylinderParams, 'openEnded').onChange(updateCylinder);
gui.add(cylinderParams, 'thetaStart', 0, 2 * Math.PI).onChange(updateCylinder);
gui.add(cylinderParams, 'thetaLength', 0, 2 * Math.PI).onChange(updateCylinder);

// Create a camera and position it
const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
camera.position.z = 5;

// Create a renderer and set its size
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);

// Add the renderer to the DOM
document.body.appendChild(renderer.domElement);

// Render the scene
function animate() {{'{'}}
    requestAnimationFrame(animate);
    // Rotate the cylinder for animation
    cylinderMesh.rotation.x += 0.01;
    cylinderMesh.rotation.y += 0.01;
    // Render the scene
    renderer.render(scene, camera);
    {{'}'}}

// Call the animate function
animate();
